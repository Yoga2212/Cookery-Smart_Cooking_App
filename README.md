# 🍳 Cookery - Smart Cooking App

## 📌 Project Overview

Cookery is a Smart Cooking Application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application helps users discover recipes, manage pantry items, create grocery lists, and plan meals efficiently.

---

## 🚀 Features

- User Registration and Login
- Personalized Recipe Recommendations
- Recipe Categories and Detailed Instructions
- Pantry Management
- Grocery List Management
- Meal Planning
- User Profile Settings
- Responsive User Interface

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Yoga2212/Cookery-Smart_Cooking_App.git
```

### Install Client Dependencies

```bash
cd client
npm install
```

### Install Server Dependencies

```bash
cd ../server
npm install
```

### Start Backend Server

```bash
npm start
```

### Start Frontend Application

```bash
cd ../client
npm start
```
